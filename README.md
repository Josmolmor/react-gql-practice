# Stack

- React
- React router
- Typescript
- Vite
- SWC
- GraphQL
- Tailwind
- shadcn
- Eslint
- Prettier

# Features

- GraphQL queries and caching using "https://countries.trevorblades.com"
- Searchable list
- Deep linking
- Dark mode (no FUOC)
- Loading status with skeletons
